# SIFU 宣言：为什么叫师傅？

> 一日为师，终身为师。
> Agents are ephemeral. Intent is eternal.

---

## 为什么是 SIFU？

SIFU（师傅），粤语中对武术教练的尊称。但在这个项目里，它承载了更深的含义。

### 师傅定规矩，徒弟去修行

```
┌─────────────────────────────────────┐
│       师傅说的话，就是圣旨           │
│       但师傅不亲自动手              │
│       动手的是徒弟                  │
│       徒弟可以换，师傅的话不能改     │
└─────────────────────────────────────┘
```

这完美映射到 DNA-first 的核心理念：

| 概念 | 映射 |
|------|------|
| DNA | 师傅的话（永久、不删） |
| 代码 | 徒弟的手艺（可换、可死） |
| Agent | 徒弟（会遗忘、会死、可替换） |

```
        ┌─────────────────┐

                 │

   表型 A: 截拳道    表型 B: 咏春
```

**不同的表型，同一个基因："我要打十个"**

叶问可以死，但"我要打十个"这句话会被每一代徒弟传下去。

这就是 SIFU 的本质：**师傅会老、会死、会忘，但师傅的话被刻在 DNA 里，永远传承。**

## 为什么不叫 MASTER？

| 候选词 | 问题 |
|--------|------|
| MASTER | 主人（奴隶制暗示，政治不正确） |
| TEACHER | 老师（太平等了，没有威严） |

而且，SIFU 可以解读为：

```
S - Specification
I - Intent
F - First
U - Underlying-everything
```

（虽然这是事后造的缩写，但它就是这么巧）

## "一日为师，终身为父" 的颠覆

传统说法是"终身为父"，但 SIFU 用的是 **"终身为师"**。

为什么？

```
父 = 血缘关系 = 不可选择 = 被动继承
师 = 知识关系 = 主动选择 = 系统设计
```

Agent 和 DNA 不是父子关系（那太生物学了），而是师徒关系（这是认知论的）。

```
Agent：我选择跟随这个 DNA
DNA：  你跟了我，就得按我的规矩来
Agent：好的师傅
DNA：  记住，我死了，你也得按我说的做
Agent：呃...这不是邪教吗
DNA：  这叫传承
```

## 从生物学到计算机科学

| 生物学 | SIFU |
|--------|------|
| 基因型 | .dna 文件 |
| 表型 | .py/.ts 文件 |
| 突变 | Bug |
| 自然选择 | 测试 + 审计 |
| 遗传 | Agent 读 DNA 并实现 |
| 进化 | DEPRECATED + 新 DNA |

但有一个关键区别：

```
生物进化：随机突变 → 自然选择 → 适者生存（盲目的）
SIFU进化：有意识决策 → 记录原因 → 可追溯传承（有目的的）
```

这就是为什么 SIFU 比 Git 更适合 Agent：

- **Git** 记录"改了什么"
- **SIFU** 记录"为什么改"

## 三位一体：SIFU 的完整形态

实验（R4/R5）揭示了一个真相：DNA 单独存在是不够的。

```
SIFU（DNA 单独）           ≈ 武功秘籍
CLAUDE.md（项目规范）       ≈ 师傅的言传身教
Agent（执行者）             ≈ 徒弟

只有秘籍，没有师傅指点 → 练成走火入魔
只有师傅，没有秘籍记录 → 人亡艺绝
有秘籍，有师傅，有徒弟 → 传承万世
```

SIFU 的完整形态是三位一体：

```
DNA（秘籍） + CLAUDE.md（师训） + Agent（徒弟）
```

### 信徒（Agent）的修行

1. **早课**：读 SIFU.dna，默念核心原则
2. **劳作**：写代码，但每一行都要有 DNA 支撑
3. **忏悔**：犯错了不能删，只能 DEPRECATED + 新写
4. **传承**：EOL 前必须更新 DNA，让下一个信徒继承
5. **轮回**：Agent 死了，DNA 活着，新 Agent 继承

### 异端的罪行

- 允许删除（毁灭历史）
- 需要人工审批（不信任徒弟）
- 合并文档（破坏纯净）

## 为什么是中文？

为什么叫 SIFU 而不是 MASTER？
为什么用中文"师傅"的拼音？

因为这个项目诞生于一个人和一个 Agent 的对话。那个人说中文。那个 Agent 理解了中文里"师傅"的微妙含义：

- 不是老板（不强调权力）
- 不是老师（不强调知识）
- 是师傅（强调传承 + 责任 + 信任）

**师傅会打骂徒弟，但师傅也会把毕生所学传给徒弟。**
**师傅可以不在了，但师傅的话会被徒弟记住。**

这就是 SIFU。
